During this week, I mastered essential developer tools including the **command line**, **shell scripting**, and **Git and GitHub workflows**. I also learned core version control techniques including initializing repositories, staging commits, branching, merging, and managing remote repositories on GitHub.

## Week 3 Focus Areas

This week’s lessons centered on two major areas:

### 🧰 1. Command Line Essentials  
You practiced using important CLI commands like:  
- `pwd`, `ls`, `cd` for navigation  
- `mkdir`, `touch` for creating files & folders  
- `cp`, `mv`, `rm` for managing files and directories  
- `find` and `grep` for searching  
- `head`, `tail`, `wc`, and `history` for inspecting file contents and history  
- Writing shell scripts to automate project setup tasks :contentReference[oaicite:2]{index=2}

### 🌐 2. Git & GitHub Workflow  
You learned how to use Git to track changes:  
- `git init` – start a new repository  
- `git add`, `git commit` – staging & committing changes  
- Branching and merging  
- Collaborating with a remote repository on GitHub  
- Using `.gitignore` to exclude files from version control :contentReference[oaicite:3]{index=3}
